# Operators

**Operators** are symbols that represent computations like addition and multiplication.

- `+` performs **addition**
- `-` performs **subtraction**
- `*` performs **multiplication**
- `/` performs **division**
- `**` performs **exponentiation**
- `%` performs **remainder (or modulus) operation**
